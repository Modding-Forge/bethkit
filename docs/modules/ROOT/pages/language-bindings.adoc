= Language Bindings (C ABI)

The `bethkit-ffi` crate exposes a stable `extern "C"` ABI that lets you call
bethkit from any language that can load a shared library and invoke C functions:
Python, C#, C++, Lua, and others.

A matching `bethkit.h` header is generated automatically by
https://github.com/mozilla/cbindgen[cbindgen] during the build and is tracked
in version control under `crates/bethkit-ffi/bethkit.h`.

== Building the shared library

[source,bash]
----
# Debug build (fast compile, debug symbols)
cargo build -p bethkit-ffi

# Release build (optimized, smaller)
cargo build -p bethkit-ffi --release
----

The output is placed in `target/debug/` or `target/release/` as:

* `bethkit_ffi.dll` / `bethkit_ffi.dll.lib` on Windows
* `libbethkit_ffi.so` on Linux
* `libbethkit_ffi.dylib` on macOS

The generated header is written to `crates/bethkit-ffi/bethkit.h`.

== Conventions

=== Return codes

Every function that can fail returns `0` on success and `-1` on error.
Functions that return a pointer signal failure by returning a null pointer.

The API distinguishes two cases when a function returns null:

Not found::
  The requested record, file, or string ID does not exist.
  The function returns null (and writes `0` into `*out_len` where applicable).
  The last error is **not** set — do not call `bethkit_last_error()`.

Error::
  A genuine failure occurred (null pointer argument, invalid UTF-8, I/O failure, etc.).
  The function returns null **and** sets the last error.

After an *error*, call `bethkit_last_error()` on the **same thread** to
retrieve a NUL-terminated description:

[source,c]
----
BethkitPlugin *p = bethkit_plugin_open("Mod.esp", BETHKIT_GAME_SKYRIM_SE);
if (!p) {
    fprintf(stderr, "bethkit error: %s\n", bethkit_last_error());
}
----

`bethkit_last_error()` returns a static empty string (never null) when no
error is set.

=== Memory ownership

The ownership model follows three rules, consistent across the entire API:

[cols="1,2,3",options="header"]
|===
| Handle type | Created by | Must be freed with

| Opaque owned handle +
  (`*BethkitPlugin`, `*BethkitArchive`, `*BethkitRecordView`, …)
| `bethkit_*_new` / `bethkit_*_open`
| Its matching `bethkit_*_free`

| Borrowed handle +
  (`*const BethkitRecord`, `*const BethkitGroup`, …)
| A getter on a parent handle
| **Never** — valid until the parent handle is freed

| Owned byte buffer +
  (`uint8_t *`, `char *` from `_as_zstring`)
| Extract or convert functions
| `bethkit_bytes_free` / the matching `*_free` function
|===

String pointers (`const char *`) returned by accessor functions are borrowed
from the handle that produced them.
They are valid until that handle is freed.

Do not call `free()` or `delete[]` on any pointer returned by bethkit.
Always use the provided `*_free` functions.

=== Thread safety

Each thread has its own error buffer.
Always check `bethkit_last_error()` on the same thread that made the failing call.
Handles themselves are not thread-safe; do not share mutable handles across
threads without external synchronization.

== Quick example — C

[source,c]
----
#include "bethkit.h"
#include <stdio.h>

int main(void) {
    /* Open a plugin */
    BethkitPlugin *plugin =
        bethkit_plugin_open("Ordinator - Perks of Skyrim.esp",
                            BETHKIT_GAME_SKYRIM_SE);
    if (!plugin) {
        fprintf(stderr, "open failed: %s\n", bethkit_last_error());
        return 1;
    }

    printf("masters: %zu\n", bethkit_plugin_master_count(plugin));

    /* Iterate groups and records */
    size_t n_groups = bethkit_plugin_group_count(plugin);
    for (size_t gi = 0; gi < n_groups; gi++) {
        const BethkitGroup *grp = bethkit_plugin_group_get(plugin, gi);
        size_t n_children = bethkit_group_child_count(grp);
        for (size_t ci = 0; ci < n_children; ci++) {
            if (!bethkit_group_child_is_record(grp, ci)) continue;
            const BethkitRecord *rec = bethkit_group_child_as_record(grp, ci);

            /* Print FormID */
            uint32_t fid = bethkit_record_form_id(rec);

            /* Print editor ID if present */
            const char *edid = bethkit_record_editor_id(rec);
            if (edid) {
                printf("  %08X  %s\n", fid, edid);
                bethkit_record_editor_id_free((char *)edid);
            }
        }
    }

    bethkit_plugin_free(plugin);
    return 0;
}
----

== Quick example — Python (ctypes)

[source,python]
----
import ctypes, sys

lib = ctypes.CDLL("bethkit_ffi.dll")        # Windows
# lib = ctypes.CDLL("./libbethkit_ffi.so")  # Linux

# bethkit_plugin_open
lib.bethkit_plugin_open.restype  = ctypes.c_void_p
lib.bethkit_plugin_open.argtypes = [ctypes.c_char_p, ctypes.c_int]

BETHKIT_GAME_SKYRIM_SE = 0

plugin = lib.bethkit_plugin_open(b"Ordinator - Perks of Skyrim.esp",
                                 BETHKIT_GAME_SKYRIM_SE)
if not plugin:
    err = ctypes.cast(lib.bethkit_last_error(), ctypes.c_char_p).value
    sys.exit(f"open failed: {err.decode()}")

# bethkit_plugin_master_count
lib.bethkit_plugin_master_count.restype  = ctypes.c_size_t
lib.bethkit_plugin_master_count.argtypes = [ctypes.c_void_p]
print("masters:", lib.bethkit_plugin_master_count(plugin))

lib.bethkit_plugin_free(plugin)
----

==== BethkitFieldValue — tagged union

Inspect the `kind` field to determine which arm of the `payload` union is active:

[cols="1,2",options="header"]
|===
| Kind | Active payload member

| `BETHKIT_FIELD_VALUE_KIND_INT` | `payload.int_val` (`int64_t`)
| `BETHKIT_FIELD_VALUE_KIND_FLOAT` | `payload.float_val` (`double`)
| `BETHKIT_FIELD_VALUE_KIND_STR` | `payload.str_val` (`const char *`, borrowed)
| `BETHKIT_FIELD_VALUE_KIND_FORM_ID` | `payload.form_id` (`uint32_t`)
| `BETHKIT_FIELD_VALUE_KIND_FORM_ID_TYPED` | `payload.form_id_typed` (`BethkitTypedFormId`)
| `BETHKIT_FIELD_VALUE_KIND_BYTES` | `payload.bytes` (`BethkitSlice`, borrowed)
| `BETHKIT_FIELD_VALUE_KIND_ENUM` | `payload.enum_val` (`BethkitEnumVal`)
| `BETHKIT_FIELD_VALUE_KIND_FLAGS` | `payload.flags_val` (`BethkitFlagsVal`)
| `BETHKIT_FIELD_VALUE_KIND_STRUCT` | `payload.struct_entries` (`BethkitFieldEntries *`, owned)
| `BETHKIT_FIELD_VALUE_KIND_ARRAY` | `payload.array_values` (`BethkitFieldValues *`, owned)
| `BETHKIT_FIELD_VALUE_KIND_LOCALIZED_ID` | `payload.localized_id` (`uint32_t`)
| `BETHKIT_FIELD_VALUE_KIND_MISSING` | — (subrecord was absent)
|===

All field values — including `Struct` and `Array` payloads — are owned by the
`BethkitRecordView` and freed automatically when `bethkit_record_view_free` is called.
Do **not** call `bethkit_field_entries_free` or `bethkit_field_values_free` on
any value obtained from a view; doing so causes a double-free.
